Web scrapping the 2 Portuguese retailers' websites

Context

This project compares prices of the 25 items included in the Food Baskets: Operational Program to Support the Neediest People between 2 portuguese retailers. To extract the data, we used the libraries Selenium, Requests, Beautiful Soup and lxml libraries. To clean and filter we used traditional data cleaning techniques and the library rapidfuzz.

  • Report
  • Jupyter notebook

Goals

  • Extract the data of the web site
  • Parse the data
  • Analyse the data

Solution

We scrapped information about 16866 products of 2 retailers. We created a data set with 16866 rows and 8 features. Then, we filtered the data set two times. Once to a data set with 3639 observations for analysis purposes. And a second to a data set with 1236 observations to make it possible to inform the consumer of the products available in both supermarkets considering the 25 Food Baskets. The approach we have followed is:

  • Data Collection
  • Data Cleaning
  • Data Visualization and Analysis
