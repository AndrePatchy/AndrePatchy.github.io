Black Friday Online Auction SQL script

Context

Black friday. The company will host a 2 weeks online auction. We've been tasked to prepare a script that supports the OLTP process.

Goals

  • Design the auction schema in a CosmosDB database

  • Create the stored procedures to support the online auction

Solution

We've built an auction schema and created stored procedures for adding products to auction, bidding on a product, removing a product from auction, updating a product’s auction status and creating lists of product’s bidding history. The index we have followed is:
