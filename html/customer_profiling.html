Customer Profiling

Context

Dig into the data and identify the variables that are crucial for distinguishing between customers and their consumption behaviors. Thoroughly analyzing the data will enable the team to identify key patterns and trends that will inform subsequent segmentation efforts.

  • Report
  • Jupyter notebook

Goals

  • Segment the company's customer database
  • Produce a comprehensive report that identifies the main customer segments, accompanied by a preliminary marketing plan that will serve as a guide for future initiatives.

Solution

Population description using statistical measures, visualizations, and business insights that reflect the most significant findings. Preprocessing techniques to transform data for clustering. Implementation of machine learning models. Customer value and customer behavior. The approach we have followed is:

  • Exploratory Data Analysis
  • Data Cleaning
  • Data Engineering
  • Feature Selection
  • Modelling

Tech Stack

  • Python (numpy, pandas, seaborn, scipy. sklearn)
  • Jupyter
  • Visual Studio Code
  • Git
  • MS Excel
  • CRISP-DM Methodology
