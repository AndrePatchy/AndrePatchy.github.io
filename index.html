The following projects are being developed in the context of 42 common-core.

The 42 Common Core was developed in Paris in 2013 and is continually evaluated and developed. Students progress through the program by completing projects and earning experience to level up, making the learning experience feel like a game.

Libft

A personal library of C functions that will help one's journey through the common-core. Libft involves learning how the standard functions of C language work by writing them from scratch and creating a personal library. This project is vital as the library will be used in future assignments at 42.

Get Next Line

A function that reads a line from a file descriptor. One learned how to read from a file descriptor using static variables. This function retrieves a single line from a specified file descriptor. When called in a loop, the function returns all the contents of the file, one line at a time until the end of the file is reached. Additionally, this function can be compiled with any buffer size specified at compile time.

Printf

This project involved recreating the well-known C library function, printf. This provided a valuable learning opportunity in variadic arguments, particularly if we intend to incorporate additional flags into our implementation of print.
